# TreeCalculator - лесной калькулятор и учёт пиломатериалов

Приложение для таксации леса, управления складом пиломатериалов и обработки заказов. Позволяет быстро вычислить объём древесины, вести ведомость с расчётом стоимости, управлять справочником пород, складскими позициями и заказчиками, а также распределять материалы со склада по заявкам.

## Основные возможности

- **Калькулятор объёма дерева** – ввод породы, диаметра и высоты, автоматический расчёт по формуле с видовым числом.
- **Ведомость** – сохранение записей в базу данных с датой, породой, объёмом и стоимостью.
- **Справочник пород** – добавление, редактирование, удаление пород с настройкой видового числа и цены за кубометр. Возможность сохранить собственный стандартный набор и сбрасывать к нему.
- **Склад пиломатериалов** – учёт досок и бруса: порода, размеры, количество, цена за штуку. Добавление, удаление позиций.
- **Заказчики** – ведение списка клиентов (название, город, адрес).
- **Заказы** – создание заказов с привязкой к заказчику и дате, добавление позиций (нужный материал и объём).
- **Распределение со склада** – для каждой позиции заказа можно подобрать материал со склада, списать нужный объём; автоматически обновляются остатки и статус выполнения.
- **Сводка по заказу** – общий объём и примерная стоимость выделенных материалов.
- **Адаптивный интерфейс** – окна с таблицами и кнопками корректно растягиваются при изменении размеров.
